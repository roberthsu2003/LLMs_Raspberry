# 目錄操作教學 – 老師示範

本檔案提供教師在課堂上示範「目錄操作」時的完整步驟與說明，包含基礎指令、常見參數與進階範例，請依序在終端機執行以下指令，並說明每一步的目的與結果。

---

## 1. 了解目前所在位置

```sh
pwd
```
*說明*：`pwd` 會印出「Present Working Directory」的完整路徑，讓學生知道自己的起點。  

---

## 2. 檢視目錄內容

```sh
# 顯示隱藏檔與詳細資訊
ls -la
```
*說明*：`-l` 提供長格式（權限、擁有者、大小、時間），`-a` 顯示以「.」開頭的隱藏檔。  

---

## 3. 建立多層目錄結構

```sh
# 一次建立 class_demo/lesson1
mkdir -p class_demo/lesson1
```
*說明*：`-p` 讓 `mkdir` 可以同時建立多層目錄，若父層已存在則不會報錯。  

---

## 4. 進入與返回目錄

```sh
# 進入新建立的目錄
cd class_demo/lesson1
pwd    # 再次確認位置

# 返回上層
cd ..
pwd    # 應該回到 class_demo

# 返回上一個工作目錄（若先前在其他目錄）
cd -
pwd
```
*說明*：`cd ..` 回到上一層，`cd -` 返回前一次所在的目錄，方便在多個目錄間切換。  

---

## 5. 列出特定類型的檔案

```sh
# 只列出 .txt 檔案
ls *.txt

# 列出目錄下的子目錄
ls -d */
```
*說明*：使用通配符 `*` 可快速篩選檔案類型，`-d */` 僅顯示目錄本身而非其內容。  

---

## 6. 移除空目錄

```sh
# 刪除先前建立的練習目錄
cd ~            # 先回到家目錄，確保不在要刪除的目錄內
rmdir -p class_demo/lesson1
```
*說明*：`rmdir` 只能刪除空目錄，`-p` 會自動移除上層目錄（若已變成空的話）。  

---

## 7. 進階範例：一次列出多層目錄的樹狀結構

```sh
# 若系統有 tree 指令，顯示目錄結構
tree -L 2
```
*說明*：`tree` 以樹狀圖方式呈現目錄層級，`-L 2` 限制深度為兩層。若未安裝，可使用 `apt install tree` 先安裝。  

---

## 8. 小結

- **`pwd`**：確認所在位置  
- **`ls`**：檢視目錄內容與篩選  
- **`mkdir`**：建立目錄（`-p` 逐層）  
- **`cd`**：切換目錄（`..`、`-`）  
- **`rmdir`**：刪除空目錄  

請依照上述步驟讓學生逐一執行，並在每一步停下來詢問「你看到什麼？」、「為什麼會這樣？」以加深理解。祝教學順利！
