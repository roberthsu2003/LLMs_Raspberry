## 語法說明：指令的組成

在 Linux 終端機中，一個完整的指令通常由三個部分組成：**命令 (Command)**、**選項 (Options/Flags)** 和 **參數 (Arguments)**。

`命令 [選項] [參數]`

### 1. 命令 (Command)
- **是什麼**：您希望電腦執行的程式或工具的名稱。它是指令的核心，不可或缺。
- **範例**：`ls`, `cd`, `grep`, `rm`

### 2. 選項 (Option / Flag)
- **是什麼**：用來調整或改變命令的預設行為。它們就像是命令的「附加功能開關」。
- **格式**：
    - **短格式 (Short Option / Flag)**：通常由一個連字號 (`-`) 加上一個字母組成，例如 `-l`, `-a`。多個短選項可以合併，例如 `-la`。
    - **長格式 (Long Option)**：由兩個連字號 (`--`) 加上一個完整的單字組成，例如 `--all`, `--human-readable`。
- **範例**：
    - `ls -l`：`-l` 選項讓 `ls` 以長格式（詳細列表）顯示。
    - `rm -i`：`-i` 選項讓 `rm` 在刪除前進行互動式確認。

### 3. 參數 (Argument)
- **是什麼**：命令要操作的對象，例如檔案名稱、目錄路徑、要搜尋的文字等。
- **範例**：
    - `cd /home/pi`：`/home/pi` 是 `cd` 命令的參數，告訴它要切換到哪個目錄。
    - `cat note.txt`：`note.txt` 是 `cat` 命令的參數，告訴它要顯示哪個檔案的內容。

### 綜合範例

讓我們來解析一個完整的指令：
`ls -la /home/pi`

- **命令 (Command)**: `ls` (列出目錄內容)
- **選項 (Options)**: `-la` ( `-l` 和 `-a` 的組合，代表「使用長格式」並「顯示所有檔案，包括隱藏檔」)
- **參數 (Argument)**: `/home/pi` (要列出內容的目錄路徑)

理解這三個組成部分，將幫助您更快速地學習和使用新的 Linux 指令。
