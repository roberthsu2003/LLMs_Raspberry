# 模型操作

**設定位置:** 管理員控制台 -> 設定 -> 連線

---

**預設模型實作**
- **預設使用 Ollama 運行模型**

> 如果無法運行,必需手動改為`http://127.0.0.1:11434`

---

- **預設有開啟Arena Model**

主要是隨機使用目前在open-Webui內所設定的模型,並且測試準確度

> 設定位置: 管理員控制台 -> 設定 -> 評估 -> 競技場模型

---

**使用OpenAI外部模型**

- 使用 Open Router 運行模型
> OpenRouter 官方 API 的 base URL 是：`https://openrouter.ai/api/v1`

---

- 使用大模型API(GPT-4,GEMINI)
> Google API 的 base URL 是：`https://generativelanguage.googleapis.com/v1beta/openai`
