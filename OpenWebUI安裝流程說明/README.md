## Open-WebUI API

- VSCode安裝THUNDER CLIENT 套件(取代curl command)

## Authentication(驗證)

- 至Settings > Account `建立一個 Bearer Token`

## 取出所有的Model
- Endpoint: `GET /api/models`

```
curl -H "Authorization: Bearer YOUR_API_KEY" http://localhost:3000/api/models
```

## Chat Completions

- Endpoint: `POST /api/chat/completions`
