## 網頁設計參考網站
### 1. 殿堂級獲獎作品（看趨勢與創新）

這些網站專門評選全球最優秀的網頁作品，適合用來觀察最新的技術趨勢和具衝擊力的視覺效果。

* **[Awwwards](https://www.awwwards.com/)**：網頁設計界的「奧斯卡」。除了看獲獎作品，還能看到專業評審對設計、創意、易用性和內容的評分。
* **[The FWA](https://thefwa.com/)**：專注於互動與前衛設計，很多結合 3D、AR 或極致動畫的作品都會在這裡出現。
* **[CSS Design Awards](https://www.cssdesignawards.com/)**：同樣是專業的獲獎平台，能找到許多在前端技術應用上非常出色的網站參考。

### 2. 綜合性設計平台（看 UI/UX 與視覺素材）

如果你不只看網頁，還想參考 UI 介面、圖示或插畫，這些平台內容最豐富。

* **[Dribbble](https://dribbble.com/)**：設計師展示「概念作品」的社群。適合尋找配色、按鈕樣式、微動效（Micro-interactions）的靈感。
* **[Behance](https://www.behance.net/)**：Adobe 旗下的平台，多為完整的設計案例（Case Studies），可以深入了解一個專案從概念到完稿的過程。
* **[suspicious link removed]**：強大的瀑布流搜尋功能，輸入「Web Design Layout」或「Landing Page」能快速產出大量視覺板塊。

### 3. 精選畫廊與實用參考（看排版與落地實作）

這些網站收集的是「真實運作中」的優質網站，設計較偏向實用、乾淨，適合找尋排版結構靈感。

* **[Siteinspire](https://www.siteinspire.com/)**：分類非常細緻（按風格、產業、平台類型），且收錄的作品大多排版簡潔、具備高水準的審美。
* **[Landbook](https://land-book.com/)**：專門收錄登陸頁（Landing Page）的畫廊，對於做行銷頁面、產品官網非常有幫助。
* **[Lapa Ninja](https://www.lapa.ninja/)**：同樣是 Landing Page 的精選站，特色是能直接下載一些免費的 UI Kit 資源。
* **[One Page Love](https://onepagelove.com/)**：專注於一頁式網頁設計（Single Page Design），非常適合需要快速呈現內容的專案。

### 4. 特定風格與特殊需求

* **日系美學**：
* **[S5-Style](https://bm.s5-style.com/)** & **[MUUUUU.ORG](https://muuuuu.org/)**：如果你喜歡日式設計的細膩、留白與精確排版，這兩個網站是最佳去處。


* **SaaS 與 App 介面**：
* **[Mobbin](https://mobbin.com/)**：如果你在做 Web App，這網站收錄了大量主流軟體的真實畫面截圖（包含登入流、設定頁等細節）。


* **局部靈感**：
* **[Footer.design](https://www.footer.design/)**：專門看網站底部的設計。
* **[Navbar Gallery](https://www.google.com/search?q=https://www.navbargallery.com/)**：專門收集各種導覽列（Navigation Bar）的樣式。



---

**建議：**
如果你是新手，建議先從 **Siteinspire** 尋找可執行的排版結構；如果是資深設計師想挑戰視覺突破，**Awwwards** 是最好的燃料。
