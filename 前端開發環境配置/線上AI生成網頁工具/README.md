# 線上AI生成網頁工具

當前比較知名、可以**用「自然語言描述想法」然後直接生成前端專案（有些還包含後端）的線上AI平台**，包括：

### 🧠 1. Lovable.dev

👉 使用自然語言描述你的網站或應用想法

- ✔ 會自動產生**完整前端（React + Tailwind CSS）**和部分後端功能（Supabase 整合）
- ✔ 支援即時預覽、客製化與 GitHub 同步
- ✔ 產出的程式碼可下載、編輯與部署

**用途：**
- 🔹 建 API + UI 的完整專案
- 🔹 不需要寫太多原始碼
- 🔹 適合快速原型或連接 Supabase

---

### 🧠 2. Co.dev

👉 透過自然語言描述產生 Next.js 全端網頁應用

- ✔ 會建置 React + Next.js 結構
- ✔ 可包含前端、資料管理與後端整合
- ✔ 可直接取得可開發與部署的專案碼

**用途：**
- 🔹 想要生成「可擴展的全端」網站
- 🔹 適合原型或 MVP 產出

---

### 🧠 3. Bolt.new

👉 Browser-based AI IDE

- ✔ 從 Prompt（自然語言）建立完整前端 + 後端專案
- ✔ 內建編輯、Node.js、React 等可運行環境
- ✔ 免費試用、有部署整合（例如 Netlify / Cloudflare）

**用途：**
- 🔹 立即在線上建立、測試與部署專案
- 🔹 想快速做全棧應用

---

### 🧠 4. MaxAI / AI Code Wizard

👉 輸入自然語言描述

- ✔ 生成 HTML / CSS / JavaScript 專案樣板程式碼
- ✔ 你可以把產出的程式碼搬到你自己的專案裡繼續開發或改造

**用途：**
- 🔹 生成前端程式碼起點
- 🔹 適合純靜態頁面或簡單互動

---

### 🧠 5. V0.dev / Scaled UI 生成工具（同屬 vibe coding）

👉 Vercel 推出的語言提示到 UI 生成

- ✔ 生成 React/Tailwind 等 UI 程式碼
- ✔ 注重前端元件與視覺布局
- ✔ 不是整個專案，但可做快速前端骨架

**用途：**
- 🔹 快速生成 UI component
- 🔹 適合配合前端框架專案

---

### 📌 選擇建議（快速判斷）

| 需求 | 推薦工具 |
|------|---------|
| 想生成完整網頁並可部署 | Lovable.dev / Co.dev |
| 想生成可立即運行的應用 | Bolt.new  |
| 想生成前端程式碼起點 | MaxAI / V0.dev / Mobirise AI |
| 想快速產生 UI 元件 | V0.dev / Framer AI / Relume |

---

### ⚠️ 注意事項

- ✔ 目前很多工具不是免費不限使用
- ✔ 生成的程式碼品質會因提示而異
- ✔ 大多數會把專案架構與元件給你，但細節仍需手動修正與部署

---
